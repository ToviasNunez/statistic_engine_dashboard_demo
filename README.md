# 🚀 Genotype Dashboard

Interactive dashboard for hybrid data simulation (AWS, On-Prem, Hybrid, Databricks).

## 🌐 Live Demo

[![View Demo](https://img.shields.io/badge/Click%20Me-View%20Dashboard-blue?style=for-the-badge)](https://toviasnunez.github.io/statistic_engine_dashboard_demo/)

## 🧠 Features

- 📊 Real-time simulation visualization
- ☁️ AWS vs On-Prem vs Hybrid comparison
- ⚡ Fast mock-based demo (no backend required)
- 📈 Charts powered by Chart.js
- 🧩 Modular React architecture

## 🛠️ Tech Stack

- React + Vite
- Chart.js
- AG Grid
- Axios
- Docker (backend version)

## 🚀 Run Locally

\`\`\`bash
npm install
npm run dev
\`\`\`

## 📦 Build

\`\`\`bash
npm run build
